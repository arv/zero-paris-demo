## Install

```
pnpm install
```

## Env

Copy/rename `.env.example` to `.env`. Set `ZERO_UPSTREAM_DB` to the connection
string of your DB.

## Start zero-cache

`pnpm zero-cache-dev`

## Start API server

The web server

```
pnpm api-server-dev
```

## Start vite

```
pnpm dev
```

## Open browser

http://localhost:5173/
